## 📚 Learning Path
1. **HTML/CSS Foundations** (Flexbox & Grid)
2. **JavaScript (ES6+)** (Async/Await, Destructuring)
3. **React Hooks** (useState, useEffect, useMemo)
4. **Next.js Routing** (App Router)
5. **UI Component Styling** (TailwindCSS)
